# MarkeT Front

Landing page desenvolvida com **React + Vite** para uma agência de gestão de marketing, com foco em performance, responsividade, apresentação de serviços e captação de leads.

## Sobre o projeto

O **MarkeT Front** é uma interface moderna para apresentação institucional de uma agência de marketing. O projeto foi criado para comunicar os serviços da empresa de forma clara, profissional e estratégica, destacando diferenciais, benefícios e chamadas para ação.

A aplicação utiliza uma estrutura baseada em componentes reutilizáveis, layout responsivo e foco em boa experiência de navegação para usuários em desktop, tablet e dispositivos móveis.

## Principais funcionalidades

- Landing page responsiva
- Interface moderna e profissional
- Seções para apresentação de serviços
- Chamadas para ação voltadas à conversão
- Estrutura componentizada
- Otimização de performance com Vite
- Layout adaptável para diferentes tamanhos de tela

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- Tailwind CSS
- CSS
- React Router DOM
- Framer Motion
- Lucide React
- Radix UI
- ESLint

## Como executar o projeto

### Pré-requisitos

Antes de começar, é necessário ter instalado:

* Node.js
* npm

### Instalação

Clone o repositório:

```bash
git clone https://github.com/Chico-wh/MarkeT---Front.git
```

Acesse a pasta do projeto:

```bash
cd MarkeT---Front
```

Instale as dependências:

```bash
npm install
```

Execute o projeto em ambiente de desenvolvimento:

```bash
npm run dev
```

Acesse no navegador:

```bash
http://localhost:5173
```

## Scripts disponíveis

Executa o projeto em modo de desenvolvimento:

```bash
npm run dev
```

Gera a versão de produção:

```bash
npm run build
```

Executa uma prévia local da build de produção:

```bash
npm run preview
```

Executa a verificação de lint:

```bash
npm run lint
```

Corrige automaticamente problemas de lint quando possível:

```bash
npm run lint:fix
```
